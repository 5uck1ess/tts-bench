# tts-bench Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a cross-platform (Windows/Mac/Linux × CPU/CUDA/MPS) speed-and-quality bench for 4 TTS models (Pocket-TTS, NeuTTS Air, NeuTTS Nano, LuxTTS) so we can pick the Cicero Sovereign-tier default.

**Architecture:** Per-model `uv` venvs + a venv-free orchestrator (`bench.py`) that shells out to per-model runner scripts via a JSON-in/JSON-out protocol. Orchestrator captures wall-time + per-core CPU + RSS externally; runner captures TTFA internally. Results emitted as CSVs + WAVs into a timestamped run dir.

**Tech Stack:** Python 3.11+, `uv` for venvs, `psutil` for monitoring, `soundfile` for WAV I/O, `pytest` for tests. PowerShell + bash install scripts. `pathlib` everywhere; no shell-string subprocesses.

**Reference docs:**
- Spec: `docs/superpowers/specs/2026-05-22-tts-bench-cross-platform-design.md`
- Source homebase doc: `homebase/wiki/research/cicero/pocket-tts-vs-neutts-bench-may2026.md`
- Model catalog: `homebase/wiki/research/deepfake/voice/new-models-apr2026.md`

**TDD discipline note:** Most orchestrator code is unit-testable (arg parsing, CSV writing, CPU polling math, resume-row matching). Those tasks follow strict TDD. Runner code calls external TTS libraries and is verified via smoke runs instead of unit tests — those tasks are marked explicitly.

"""Orchestrator for the cross-platform TTS bench.

See docs/superpowers/specs/2026-05-22-tts-bench-cross-platform-design.md.
"""

from __future__ import annotations

import argparse


ALL_MODELS = ["pocket", "neutts_air", "neutts_nano", "luxtts"]


def _csv_str(s: str) -> list[str]:
    return [item.strip() for item in s.split(",") if item.strip()]


def _csv_int(s: str) -> list[int]:
    return [int(item.strip()) for item in s.split(",") if item.strip()]


def build_parser() -> argparse.ArgumentParser:
    p = argparse.ArgumentParser(description="Cross-platform TTS bench.")
    p.add_argument(
        "--reference",
        required=True,
        help="'default' (use each model's built-in voice) or path to a reference WAV.",
    )
    p.add_argument(
        "--devices",
        type=_csv_str,
        default=None,
        help="Comma-sep list from {cpu,cuda,mps}. Default: auto-detect available.",
    )
    p.add_argument(
        "--models",
        type=_csv_str,
        default=ALL_MODELS,
        help=f"Comma-sep model list. Default: {','.join(ALL_MODELS)}.",
    )
    p.add_argument(
        "--prompts",
        type=_csv_int,
        default=None,
        help="Comma-sep prompt id list. Default: all 5.",
    )
    p.add_argument("--cold-warm-prompt", type=int, default=2)
    p.add_argument("--cold-warm-runs", type=int, default=3)
    p.add_argument("--resume", action="store_true")
    p.add_argument("--out-dir", type=str, default=None)
    p.add_argument("--timeout-s", type=int, default=60)
    return p


if __name__ == "__main__":
    args = build_parser().parse_args()
    print(args)
```

import json
import subprocess
from pathlib import Path


def parse_probe_output(stdout: str) -> dict:
    """Find the last line of stdout that parses as JSON and return it.

    Some torch builds emit warnings to stdout; the probe's JSON is always the last line.
    """
    for line in reversed(stdout.strip().splitlines()):
        line = line.strip()
        if line.startswith("{"):
            return json.loads(line)
    raise ValueError(f"No JSON line found in probe output: {stdout!r}")


def available_devices_from_probe(probe: dict) -> list[str]:
    devices = ["cpu"]
    if probe.get("cuda"):
        devices.append("cuda")
    if probe.get("mps"):
        devices.append("mps")
    return devices


def run_device_probe(model_venv_python: Path) -> dict:
    """Spawn the probe script inside a model venv and parse the result.

    The probe is in scripts/_probe_device.py. The venv python must have torch installed.
    """
    probe_script = Path(__file__).parent / "scripts" / "_probe_device.py"
    result = subprocess.run(
        [str(model_venv_python), str(probe_script)],
        capture_output=True,
        text=True,
        timeout=30,
        check=False,
    )
    return parse_probe_output(result.stdout)
```

```python
import threading
import time
from dataclasses import dataclass, field

import psutil


@dataclass
class RunnerResult:
    ok: bool
    ttfa_ms: float | None = None
    wall_s: float = 0.0
    audio_s: float | None = None
    peak_rss_mb_internal: float | None = None  # reported by runner
    peak_rss_mb_external: float = 0.0          # measured by orchestrator
    cores_active_peak: int = 0
    per_core_peaks: list[float] = field(default_factory=list)
    device_used: str | None = None
    error: str | None = None
    raw_stdout: str = ""
    raw_stderr: str = ""


def _monitor_process(proc: psutil.Process, snapshots: dict, stop: threading.Event) -> None:
    """Poll RSS and per-core CPU every 50ms until `stop` is set or the process exits."""
    peak_rss = 0.0
    per_core_peak: list[float] = []
    cores_active_peak = 0
    while not stop.is_set():
        try:
            rss = proc.memory_info().rss / (1024 * 1024)
            peak_rss = max(peak_rss, rss)
        except (psutil.NoSuchProcess, psutil.AccessDenied):
            break
        per_core = psutil.cpu_percent(interval=None, percpu=True)
        if not per_core_peak:
            per_core_peak = list(per_core)
        else:
            for i, v in enumerate(per_core):
                if i < len(per_core_peak):
                    per_core_peak[i] = max(per_core_peak[i], v)
        active_now = sum(1 for v in per_core if v > 50.0)
        cores_active_peak = max(cores_active_peak, active_now)
        time.sleep(0.05)
    snapshots["peak_rss_mb"] = peak_rss
    snapshots["per_core_peaks"] = per_core_peak
    snapshots["cores_active_peak"] = cores_active_peak


def invoke_runner(
    runner_python: str,
    runner_script: str,
    job_path: str,
    out_path: str,
    timeout_s: int,
    extra_args: list[str] | None = None,
) -> RunnerResult:
    cmd = [runner_python, runner_script, "--job", job_path, "--out", out_path]
    if extra_args:
        cmd += extra_args

    # Prime psutil.cpu_percent so the first reading is meaningful.
    psutil.cpu_percent(interval=None, percpu=True)

    start = time.perf_counter()
    try:
        proc = subprocess.Popen(
            cmd,
            stdout=subprocess.PIPE,
            stderr=subprocess.PIPE,
            text=True,
        )
    except OSError as e:
        return RunnerResult(ok=False, error=f"spawn failed: {e}")

    snapshots: dict = {}
    stop = threading.Event()
    try:
        ps_proc = psutil.Process(proc.pid)
        monitor = threading.Thread(
            target=_monitor_process,
            args=(ps_proc, snapshots, stop),
            daemon=True,
        )
        monitor.start()
    except psutil.NoSuchProcess:
        monitor = None  # process already exited; skip monitoring

    try:
        stdout, stderr = proc.communicate(timeout=timeout_s)
    except subprocess.TimeoutExpired:
        proc.kill()
        try:
            stdout, stderr = proc.communicate(timeout=5)
        except subprocess.TimeoutExpired:
            stdout, stderr = "", ""
        stop.set()
        if monitor:
            monitor.join(timeout=1)
        wall = time.perf_counter() - start
        return RunnerResult(
            ok=False,
            wall_s=wall,
            error=f"timeout after {timeout_s}s",
            raw_stdout=stdout or "",
            raw_stderr=stderr or "",
            peak_rss_mb_external=snapshots.get("peak_rss_mb", 0.0),
            per_core_peaks=snapshots.get("per_core_peaks", []),
            cores_active_peak=snapshots.get("cores_active_peak", 0),
        )

    wall = time.perf_counter() - start
    stop.set()
    if monitor:
        monitor.join(timeout=1)

    # Parse the runner's JSON result (last JSON line in stdout).
    try:
        parsed = parse_probe_output(stdout)  # reuse JSON-line picker
    except ValueError:
        return RunnerResult(
            ok=False,
            wall_s=wall,
            error="failed to parse runner JSON",
            raw_stdout=stdout,
            raw_stderr=stderr,
            peak_rss_mb_external=snapshots.get("peak_rss_mb", 0.0),
            per_core_peaks=snapshots.get("per_core_peaks", []),
            cores_active_peak=snapshots.get("cores_active_peak", 0),
        )

    if not parsed.get("ok"):
        return RunnerResult(
            ok=False,
            wall_s=wall,
            error=parsed.get("error", "runner reported not ok"),
            raw_stdout=stdout,
            raw_stderr=stderr,
            peak_rss_mb_external=snapshots.get("peak_rss_mb", 0.0),
            per_core_peaks=snapshots.get("per_core_peaks", []),
            cores_active_peak=snapshots.get("cores_active_peak", 0),
        )

    return RunnerResult(
        ok=True,
        ttfa_ms=parsed.get("ttfa_ms"),
        wall_s=wall,
        audio_s=parsed.get("audio_s"),
        peak_rss_mb_internal=parsed.get("peak_rss_mb"),
        peak_rss_mb_external=snapshots.get("peak_rss_mb", 0.0),
        cores_active_peak=snapshots.get("cores_active_peak", 0),
        per_core_peaks=snapshots.get("per_core_peaks", []),
        device_used=parsed.get("device_used"),
        raw_stdout=stdout,
        raw_stderr=stderr,
    )
```

    with path.open("w", newline="", encoding="utf-8") as f:
        writer = _csv.DictWriter(f, fieldnames=QUALITY_CSV_COLUMNS)

                "artifacts_1_5": "",
                "notes": "",
            })


def build_job_dict(
    prompt: str,
    reference: str,
    device: str,
    model_variant: str | None,
    warmup: bool,
) -> dict:
    """Build the job.json payload for a runner.

    `reference` is either the string 'default' or a path to a WAV. If a WAV, a sibling
    .txt file with the same stem must exist (the transcript).
    """
    if reference == "default":
        reference_wav = None
        reference_txt = None
    else:
        wav_path = Path(reference)
        txt_path = wav_path.with_suffix(".txt")
        if not txt_path.exists():
            raise FileNotFoundError(
                f"reference .txt transcript not found: {txt_path}. "
                f"Place a transcript next to {wav_path}."
            )
        reference_wav = str(wav_path)
        reference_txt = str(txt_path)
    return {
        "prompt": prompt,
        "reference_wav": reference_wav,
        "reference_txt": reference_txt,
        "device": device,
        "model_variant": model_variant,

@dataclass
class ModelConfig:
    name: str                       # e.g. "pocket_cpu"
    device: str                     # "cpu" | "cuda" | "mps"
    runner_python: str              # path to the runner venv python
    runner_script: str              # path to runners/<x>_runner.py
    supports_multilingual: bool = False
    model_variant: str | None = None     # neutts: "air" | "nano"
    extra_args: list[str] = field(default_factory=list)


def _audio_duration_s(wav_path: Path) -> float | None:
    try:
        import soundfile as sf
        with sf.SoundFile(str(wav_path)) as f:
            return len(f) / f.samplerate
    except Exception:
        return None


def _row_for_result(
    *, platform_name, device, model_config_name, prompt_id, result: RunnerResult,
    cold_or_warm, reference_kind, commit, run_started_at, audio_s_from_wav,
) -> dict:
    # Prefer the runner-reported audio_s; fall back to a soundfile measurement.
    audio_s = result.audio_s if result.audio_s is not None else audio_s_from_wav
    rtf = (audio_s / result.wall_s) if (audio_s and result.wall_s) else None
    return {
        "platform": platform_name,
        "device": device,
        "model_config": model_config_name,
        "prompt_id": prompt_id,
        "ttfa_ms": result.ttfa_ms if result.ttfa_ms is not None else "",
        "wall_s": round(result.wall_s, 4),
        "audio_s": round(audio_s, 4) if audio_s else "",
        "rtf": round(rtf, 4) if rtf else "",
        "peak_rss_mb": round(max(result.peak_rss_mb_external, result.peak_rss_mb_internal or 0), 1),
        "cores_active_peak": result.cores_active_peak,
        "per_core_peaks": "[" + ",".join(f"{v:.0f}" for v in result.per_core_peaks) + "]",
        "cold_or_warm": cold_or_warm,
        "reference_kind": reference_kind,
        "ok": result.ok,
        "error": (result.error or "")[:200],
        "commit": commit,
        "run_started_at": run_started_at,
    }

#Requires -Version 5.1
$ErrorActionPreference = "Stop"

function Install-Venv {
    param(
        [string]$Name,
        [string]$RequirementsFile,
        [scriptblock]$ExtraSteps = $null
    )
    $VenvPath = Join-Path $RepoRoot "venvs\$Name"
    if (-not (Test-Path $VenvPath)) {
        Write-Host "Creating venv: $VenvPath" -ForegroundColor Cyan
        uv venv $VenvPath --python 3.11

        Write-Host "Venv already exists: $VenvPath" -ForegroundColor Gray
    }
    $Py = Join-Path $VenvPath "Scripts\python.exe"
    if (Test-Path $RequirementsFile) {
        & $Py -m pip install -r $RequirementsFile
    }
    if ($ExtraSteps) {
        & $ExtraSteps $Py
    }
    return $Py
}

$Status = @{}

# 1. Orchestrator
Write-Host "`n=== Orchestrator ===" -ForegroundColor Yellow
try {

#!/usr/bin/env bash

REPO_ROOT="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"

install_venv() {
    local name="$1"
    local req="$2"
    local venv="$REPO_ROOT/venvs/$name"
    if [ ! -d "$venv" ]; then
        echo "Creating venv: $venv"
        uv venv "$venv" --python 3.11
    else
        echo "Venv already exists: $venv"
    fi
    local py="$venv/bin/python"
    if [ -f "$req" ]; then
        "$py" -m pip install -r "$req"
    fi
    echo "$py"
}

echo
echo "=== Orchestrator ==="
if install_venv orchestrator requirements-orchestrator.txt > /dev/null; then
    STATUS[orchestrator]="ok"
else
    STATUS[orchestrator]="FAILED"
fi

echo
echo "=== Pocket-TTS ==="

    SRC="$REPO_ROOT/venvs/pocket/src"
    [ -d "$SRC" ] || git clone https://github.com/kyutai-labs/pocket-tts "$SRC"
    "$PY" -m pip install -e "$SRC" && STATUS[pocket]="ok" || STATUS[pocket]="FAILED"
else
    STATUS[pocket]="FAILED"
fi

echo
echo "=== NeuTTS Air + Nano ==="
if install_venv neutts requirements-neutts.txt > /dev/null; then
    STATUS[neutts]="ok"
else
    STATUS[neutts]="FAILED"
fi

echo
echo "=== LuxTTS ==="
if install_venv luxtts requirements-luxtts.txt > /dev/null; then
    STATUS[luxtts]="ok"
else
    STATUS[luxtts]="FAILED"
fi

echo
echo "=== Install summary ==="
for k in "${!STATUS[@]}"; do
